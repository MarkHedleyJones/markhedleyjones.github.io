Interfacing sensors and actuators with ROS allows these devices to be integrated into larger systems easily. This project demonstrates the use of a Bosch Rexroth IndraDriveCS motor controller and servo motor via a ROS interface. The ROS control node connects to the IndraDrive over EtherNet/IP, which is an industrial control protocol commonly used in PLCs. This node is available on GitHub.
